## III. 핵심 컴포넌트별 사용 규격
### 1. 공식/수치 분석 컴포넌트 (Formula Component - $E_{available} = E_{initial} - \sum C_{others}$)
*   **배치 위치:** 반드시 영역 B의 중앙에 배치하여 시선의 초점을 맞춥니다.
*   **색상 사용:** 기본 텍스트는 블랙(#333333)을 유지하며, **핵심 변수(예: $E_{initial}$)**에는 강조색 `#FF6B6B`를 적용합니다.
*   **레이어 분리 지침 (Editor 필수):** 수식 자체 레이어 / 각 변수명 레이어 / 설명 텍스트 레이어 3개로 완벽히 분리되어야 합니다.

### 2. 체크리스트 컴포넌트 (Action Checklist Component)
*   **목적:** 독자가 즉시 메모하고 실천할 수 있도록 '구조화된 자산'으로 활용합니다.
*   **배치 위치:** 반드시 영역 C의 최상단에 배치하여 가장 눈에 띄게 만듭니다.
*   **디자인 규칙:** 항목별로 체크박스(☐)를 명확히 사용하고, 전체 리스트는 Mint Green 계열 배경색(#D9EAD3)이 은은하게 적용된 박스 안에 수록되어야 합니다. (최대 5개 항목 제한).

### 3. 키워드 강조 컴포넌트 (Concept Highlight Box)
*   **목적:** 복잡한 개념을 한 문장으로 요약하여 독자의 기억에 남도록 합니다.
*   **디자인 규칙:** 배경은 Blue-Grey 계열(#B0C4DE)의 연한 그라데이션을 사용하고, 폰트 크기를 본문보다 2단계 크게 설정합니다. (제목 역할 수행).
